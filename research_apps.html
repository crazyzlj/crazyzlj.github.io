Coding

I read/use/write cross-platform algorithms and applications with Python, C/C++, and a little Fortran, especially high-performance programming of C++ and Fortran with MPI and OpenMP.

The list of my favorite or customized softwares/tools/websites/tool-chains is as follows:

  • VSCode for reading and writing almost everything related to plaintext format

  • CMake for building C++ and Fortran programs

  • Git, Github, BitBucket, and SourceForge for version control of source code

  • PyCharm together with Anaconda Distribution for writing Python programs compatible with Python2.7 and Python3

  • CLion together with MSYS2/MinGW-w64 on Windows 10 and GCC on Linux or macOS for C++ and Fortran

  • Visual Studio with ReSharper and Intel Parallel Studio XE for C++ and Fortran

Normally, I develop code on VS2015/VS2010, test on MinGW-w64, Intel Parallel Studio XE, and Xcode, and run on Linux cluster using GCC or Intel C compilers.
